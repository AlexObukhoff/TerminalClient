------------------------------------------------------------
TerminalClient 3.0.0 ReadMe
------------------------------------------------------------

Уважаемый пользователь,

в этом файле содержится важная информация об этой программе.

TerminalClient 3.0.0 - программное обеспечения для терминалов самообслуживания, 
разработанное компанией Киберплат. Данное программное обеспечение осуществляет 
прием и проведение платежей в пользу провайдеров, подключенных к системе процессинга 
компании Киберплат.

------------------------------------------------------------
Системные требования:
------------------------------------------------------------

- Microsoft Windows XP SP3, Windows 7 SP1;
- Pentium-совместимый компьютер (рекомендуется от Intel Celeron G18*/AMD Athlon II);
- 1024 МБ ОЗУ или выше;
- от 200 МБ свободного дискового пространства;
- Разрешение монитора 1280x1024 16млн. цветов

------------------------------------------------------------
Компоненты программы
------------------------------------------------------------

Программный комплекс состоит из следующих модулей:
	- client.exe  - основной модуль ПО;
	- guard.exe   - модуль контроля приложений;
	- updater.exe - модуль обновления;
	- tray.exe   - приложение для панели задач Windows для запуска/останова терминального ПО.

Запуск ПО осуществляется с помощью модуля контроля приложений (guard.exe), вручную или с 
помощью иконки в панели задач Windows.

------------------------------------------------------------
Настройка программы
------------------------------------------------------------

После запуска сервиса контроля приложений Вы попадете на окно первоначальной настройки приложения,
где по шагам будет предложено настроить терминал. 

Модемное соединение создается автоматически, выбирается из списка уже существующих или из шаблона.

После генерации ключей, терминал ожидает их заверения в кабинете дилера, после чего загрузит с сервера мониторинга 
конфигурацию и перейдет в рабочий режим.

Настройка бизнес правил терминала осуществяются через Web-мониторинг Киберплат https://portal.cyberplat.ru/

Если у Вас не подключен принтер или купюроприемник, то установите в профиле мониторинга опции игнорирования ошибок этих устройств,
после чего отправьте конфигурацию на терминал.

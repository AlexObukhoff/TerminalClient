1. Выбор оператора

Нажмите на кнопку с изображением символики искомого поставщика услуг (а)
(сотового оператора, оператора кабельного телевидения, интернет провайдера и т.п.).

2. Ввод данных

При помощи сенсорной клавиатуры (а) введите запрашиваемые данные
(номер телефона, лицевого счета, договора или другой идентификатор клиента).
По окончанию ввода нажмите появившуюся кнопку «Далее» (б).

3. Внесение средств

Вставьте купюру в купюроприемник, дождитесь появления внесенной суммы на экране (а). Если купюр несколько, вставляйте их по одной.
После внесения нужной суммы нажмите кнопку «Оплатить» (б).

4. Платеж завершен

Дождитесь появления на экране сообщения о принятии платежа и возьмите квитанцию об оплате.
Будьте внимательны, аппарат сдачи не выдает!
